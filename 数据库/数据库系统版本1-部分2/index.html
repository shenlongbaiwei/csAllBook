    复杂查询- Some与All子查询¶

    例子：

    找出工资最低的教师姓名

    SQL
    SELECT TNAME 
FROM TEACHER
WHERE SALARY<=ALL ( SELECT SALARY FROM TEACHER);

    找出001号课程成绩不是最高的所有学生的学号

    SQL
    SELECT SNO
FROM SC
WHERE CNO= '001' AND SCORE < SOME(SELECT SCORE FROM SC WHERE CNO='001');

    找出所有课程都不及格的学生姓名

    SQL
    SELECT SNAME
FROM STUDENT 
WHERE 60> ALL(SELECT SCORE FROM SC WHERE SNO=STUDENT.SNO);

    找出001号课程成绩最高的所有学生的学号

    SQL
    SELECT SNO FROM SC
WHERE CNO='001' AND SCORE >= ALL(SELECT SCORE FROM SC WHERE CNO='001');

    找出98030101号同学成绩最低的课程号

    SQL
    SELECT CNO FROM SC
WHERE SNO='98030101' AND SCORE <= ALL (SELECT SCORE FROM SC WHERE SNO='98030101');

    找出张三同学成绩最低的课程号

    SQL
    SELECT CNO
FROM SC,STUDENT S
WHERE SNAME='张三' AND S.SNO=SC.SNO AND SCORE <=ALL(SELECT SCORE FROM SC WHERE SNO=S.SNO);

    需要注意的¶

    这个地方的<>some可能找出来的结果不全，因此他是和其余两个不等价的。

    利用SQL语言进行结果计算与聚集计算¶

    1、结果计算¶

    求解有差额的任意两位教师的薪水差额，差额>0

    SQL
    SELECT T1.TNAME AS TR1,T2.TNAME AS TR2,T1.SALARY-T2.SALARY
FROM TEACHER T1,TEACHER T2
WHERE T1.SALARY>T2.SALARY;

    依据学生年龄求学生的出生年份，当前是2024年

    SQL
    SELECT S.SNO,S.SNAME,2024-S.SAGE+1 AS SYEAR
FROM STUDENT S;

    2、应用聚集函数进行统计计算¶

    求教师的工资总额

    SQL
    SELECT SUM(SALARY) FROM TEACHER;

    求计算机系教师的工资总额

    SQL
    SELECT SUM(SALARY) FROM TEACHER T,DEPT WHERE DEPT.DNAME='计算机' AND DEPT.DNO=T.DNO;

    球数据库课程的平均成绩

    Text Only
    SELECT AVG(SCORE)
FROM COURSE C,SC
WHERE C.CNAME='数据库' AND C.CNO=SC.CNO;
