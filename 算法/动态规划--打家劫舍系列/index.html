    动态规划--打家劫舍系列¶

    本文主要是对代码随想录的学习总结，仅用于个人学习总结复习。

    打家劫舍1¶

    题目来源：https://leetcode.cn/problems/house-robber/

    思路：

    其实这类题目的关键都是某个屋子偷与不偷的区别，并且在满足题目要求的情况下，寻找最大金额。

    动态规划五部曲：

    1、确定dp数组的定义，dp[i]表示考虑房屋序号小于i的房屋，最大可以偷的金额。

    2、dp数组的递推公式：dp数组只有偷不偷的选择，因此，dp[i]=max(dp[i-2]+nums[i],dp[i-1]);

    3、dp数组初始化：dp的推导公式是取决于i-1，i-2，因此初始化dp[0]=0,dp[1]=nums[0];

    4、确定遍历顺序：dp[i]的取值取决于前两个元素，因此遍历顺序是从前向后。

    5、举例验证。

    代码：

    C++
    int rob(vector<int>& nums) {
         if (nums.size() == 0) return 0;
        if (nums.size() == 1) return nums[0];
        vector<int> dp(nums.size(),0);
        dp[0]=nums[0];
        dp[1]=max(nums[1],nums[0]);
        //遍历dp数组
        for(int i=2;i<dp.size();i++){
            dp[i]=max(dp[i-2]+nums[i],dp[i-1]);
        }
        return dp[nums.size()-1];
    }

    打家劫舍2¶

    题目来源：https://leetcode.cn/problems/house-robber-ii/

    思路：该题和前面那个题的区别是首尾相连，因此该题只有三种情况。

    • 只取除了首尾两个元素的中间的屋子的钱
    • 只取首端到nums.size()-2中间的位置的屋子的钱
    • 取除了首端之外的所有房屋的最高金额。

    其实后两种情况都包含第一种情况，因此只需要考虑后两种情况即可。

    剩余思路和前面题目的思路类似，主要区别是我们需要将两种情况都进行计算并进行对比，在计算其中一种情况的时候，只需要计算和前面一题一样的部分即可。

    代码：

    C++
    int rob(vector<int>& nums) {
        if (nums.size() == 0) return 0;
        if (nums.size() == 1) return nums[0];
        int result1 = robRange(nums, 0, nums.size() - 2); // 情况二
        int result2 = robRange(nums, 1, nums.size() - 1); // 情况三
        return max(result1, result2);
    }
    // 198.打家劫舍的逻辑
    int robRange(vector<int>& nums, int start, int end) {
        if (end == start) return nums[start];
        vector<int> dp(nums.size());
        dp[start] = nums[start];
        dp[start + 1] = max(nums[start], nums[start + 1]);
        for (int i = start + 2; i <= end; i++) {
            dp[i] = max(dp[i - 2] + nums[i], dp[i - 1]);
        }
        return dp[end];
    }

    打家劫舍3¶

    题目来源：https://leetcode.cn/problems/house-robber-iii/

    该题的思路是：

    1、需要解决的问题是树形动态规划。

    2、主要考虑的是当前结点是否偷，如果选择当前节点，则当前节点的左右子节点就无法选择。如果不选择当前节点，则左右节点都可以选择。最后对比找到最大金额，其他地方和前面类似。

    3、确定递归函数的参数和返回值。

    4、确定终止条件

    5、递归体，遍历顺序

    6、确定单层递归的逻辑

    7、注意遍历顺序，该题的遍历顺序选择后序遍历，即左右根。

    代码：

    C++
    int rob(TreeNode* root) {
        vector<int> result = robTree(root);
        return max(result[0], result[1]);
    }
    // 长度为2的数组，0：不偷，1：偷
    vector<int> robTree(TreeNode* cur) {
        if (cur == NULL) return vector<int>{0, 0};
        vector<int> left = robTree(cur->left);
        vector<int> right = robTree(cur->right);
        // 偷cur，那么就不能偷左右节点。
        int val1 = cur->val + left[0] + right[0];
        // 不偷cur，那么可以偷也可以不偷左右节点，则取较大的情况
        int val2 = max(left[0], left[1]) + max(right[0], right[1]);
        return {val2, val1};
    }

    总结¶

    上述三个题目主要是利用动态规划，其中还有考虑到树形递归，首尾相连递归和普通递归，整体上就是这三种类型。
