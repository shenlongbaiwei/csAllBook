    动态规划—0-1背包问题¶

    学习代码随想录相关动态规划的总结

    用一维动态数组¶

    C++
    #include<iostream>
#include<vector>
using namespace std;
int main(){
    int M,N;
    //N是背包容量
    cin>>M>>N;
    //初始化cost,weight
    vector<int> cost(M);
    vector<int> values(N);
    for(int i=0;i<M;i++){
        cin>>cost[i];
    }
    for(int i=0;i<M;i++){
        cin>>values[i];
    }
    //遍历dp数组，并进行相关的初始化和推导
    vector<int> dp(N+1,0);
    for(int i=0;i<M;i++){
        for(int j=N;N>=cost[i];N--){
            dp[j]=max(dp[j],dp[j-cost[i]]+value[i]);
        }
    }
    cout<<dp[N]<<endl;
    return 0;
}

    该方法是使用一维数组来进行相关的dp数组遍历，实现0-1背包的动态规划。

    整体思路主要点是dp数组的初始化，和对应数组的下标和内容的理解，以及dp数组的推导

    用二维动态数组¶

    代码：

    引用自carl的代码随想录：https://programmercarl.com/%E8%83%8C%E5%8C%85%E7%90%86%E8%AE%BA%E5%9F%BA%E7%A1%8001%E8%83%8C%E5%8C%85-1.html#%E6%80%9D%E8%B7%AF

    C++
    //二维dp数组实现
#include <bits/stdc++.h>
using namespace std;

int n, bagweight;// bagweight代表行李箱空间
void solve() {
    vector<int> weight(n, 0); // 存储每件物品所占空间
    vector<int> value(n, 0);  // 存储每件物品价值
    for(int i = 0; i < n; ++i) {
        cin >> weight[i];
    }
    for(int j = 0; j < n; ++j) {
        cin >> value[j];
    }
    // dp数组, dp[i][j]代表行李箱空间为j的情况下,从下标为[0, i]的物品里面任意取,能达到的最大价值
    vector<vector<int>> dp(weight.size(), vector<int>(bagweight + 1, 0));

    // 初始化, 因为需要用到dp[i - 1]的值
    // j < weight[0]已在上方被初始化为0
    // j >= weight[0]的值就初始化为value[0]
    for (int j = weight[0]; j <= bagweight; j++) {
        dp[0][j] = value[0];
    }

    for(int i = 1; i < weight.size(); i++) { // 遍历科研物品
        for(int j = 0; j <= bagweight; j++) { // 遍历行李箱容量
            // 如果装不下这个物品,那么就继承dp[i - 1][j]的值
            if (j < weight[i]) dp[i][j] = dp[i - 1][j];
            // 如果能装下,就将值更新为 不装这个物品的最大值 和 装这个物品的最大值 中的 最大值
            // 装这个物品的最大值由容量为j - weight[i]的包任意放入序号为[0, i - 1]的最大值 + 该物品的价值构成
            else dp[i][j] = max(dp[i - 1][j], dp[i - 1][j - weight[i]] + value[i]);
        }
    }
    cout << dp[weight.size() - 1][bagweight] << endl;
}

int main() {
    while(cin >> n >> bagweight) {
        solve();
    }
    return 0;
}

    整体思路都是要注意dp数组的初始化，含义，递推公式和递推顺序。

    分割等和子集¶

    题目来源：

    https://leetcode.cn/problems/partition-equal-subset-sum/description/

    思路：

    将本题理解成0-1背包问题，并将相关内容进行转换，套用到0-1背包问题上的解决方法上来。

    其实就是找到集合里面出现总和/2的子集总和，只要找到了，就说明能够分成相同的子集，反之则不能。

    0-1背包问题的模板¶

    1. 确定dp数组含义，dp[i]表示背包最大容量为i，且dp[i]代表当前容量的背包的物品的价值
    2. 确定dp递推公式：dp[i]=max(dp[i],dp[i-weight[j]]+values[j])
    3. 初始化，一般都构建一个最大容量的数量的dp数组的size，初始化为0
    4. 递推顺序，二维：外层背包容量，内层遍历物品
    5. 举例推导：自行推导。

    套用模板¶

    本题很明显需要找到是否在target为sum/2的时候装满。

    1. dp数组，定义为一维数组，dp[i]表示，当前背包能填进去最大总和为i的数字集合的和，i表示当前能容纳的最大的数字和
    2. dp递推公式：dp[i]=max(dp[i],dp[i-cost[j]]+weight[j])，该题中，每个数字的容量和对应的价值都相同，即cost[i]=weight[i]，而当前每一步都存在取不取当前的数字，选择不取就是原来的值，选择取则是当前前面的数组dp减去要取的元素的最大和再加上当前取的元素的大小
    3. 初始化：cost，weight，直接初始化为题目默认的函数的元素大小,dp数组初始化为0即可，数组大小为所有数组大小总和+1即可
    4. 遍历顺序：一维数组，所以先遍历物品，再遍历背包
    5. 自行举例推导

    代码：

    C++
    bool canPartition(vector<int>& nums) {
        int sum=0;
        for(int i=0;i<nums.size();i++){
            sum+=nums[i];
        }
        vector<int> dp(10001,0);
        if(sum%2==1) return false;
        int target_1=sum/2;
        //0-1背包循环即可
        for(int i=0;i<nums.size();i++){
            for(int target=target_1;target>=nums[i];target--){
                dp[target]=max(dp[target],dp[target-nums[i]]+nums[i]);
            }
        }
        //如果存在dp[target_1]==target_1，则返回true
        if(target_1==dp[target_1]) return true;
        return false;

    }

    最后一块石头的重量Ⅱ¶

    题目来源：

    https://leetcode.cn/problems/last-stone-weight-ii/

    思路：此题和上面题的思路相同，都是套用0-1背包理论进行推导即可。

    0-1模板：见上一题

    套用模板：与上一题相似，只不过要注意dp数组的含义，dp[i]表示容量为i的背包，最多能够背的重量为dp[i]表示。

    “最多可以装的价值为 dp[j]” == “最多可以背的重量为dp[j]”。

    其他部分都相似。

    其实求解最后剩余石头大小，其实就是将石头进行总重量除以二进行对称碰撞消除，最后剩余的石头大小重量，即

    sum-dp[target] - dp[target]. target=sum/2.target是向下取整， dp[target]<sum/2。

    代码：

    C++
    int lastStoneWeightII(vector<int>& stones) {
        vector<int> dp(15001,0);
        int sum=0;
        for(int i=0;i<stones.size();i++) sum+=stones[i];
        int target=sum/2;
        //遍历石头
        for(int i=0;i<stones.size();i++){
            //遍历装石头的背包
            for(int j=target;j>=stones[i];j--){
                dp[j]=max(dp[j],dp[j-stones[i]]+stones[i]);
            }
        }
        //返回结果
        return (sum-dp[target])-dp[target];
    }

    上述部分即可得当前石头消除之后的得到的石头大小。
