动态规划合集

本合集所有题目均来自leetcode，相关解法是对代码随想录carl老师思路的个人总结和提炼，仅用于学习途径。

递归五公式：

1、确定dp数组含义及其下标含义

2、确定dp数组递推公式

2、初始化dp数组

3、确定遍历顺序

4、举例进行推导dp数组

斐波那契数列

题目要求很简单，无需多言。

1、dp数组表示对应元素大小

2、递推公式:dp[i]=dp[i-1]+dp[i-2];

3、初始化dp数组：dp[0]=0,dp[1]=1;

4、确定遍历顺序：很明显是从前往后进行遍历，从0,1-》2，3.。。

5、直接进行遍历推导呗

代码：

int fib(int n){
    if(n<=1) return n;
    int dp[2];
    dp[0]=0;
    dp[1]=1;
    for(int i=2;i<=n;i++){
        int sum=dp[0]+dp[1];
        dp[0]=dp[1];
        dp[1]=sum;
}
    return dp[1];

}

爬楼梯

思路：

爬楼梯问题，其实就是初始dp数组，然后dp数组的递推公式如下：

dp[i]=dp[i-1]+dp[i-2];

递推顺序：很明显从前往后。

然后就是for循环进行递推遍历咯。

代码：

int climbStairs(int n) {
        if(n<=1) return n;
        int dp[3];
        dp[1]=1;
        dp[2]=2;
        for(int i=3;i<=n;i++){
            int sum=dp[1]+dp[2];
            dp[1]=dp[2];
            dp[2]=sum;
        }
        return dp[2];
    }

总结：

以上就是相关爬楼梯问题的解法，其实这个方法思路也可以应用到，爬一层楼一次一步，，一次两步，一次三步，一直往下举个例子。这种也都能进行动态规划。

class Solution {
public:
    int climbStairs(int n) {
        vector<int> dp(n + 1, 0);
        dp[0] = 1;
        for (int i = 1; i <= n; i++) {
            for (int j = 1; j <= m; j++) { // 把m换成2，就可以AC爬楼梯这道题
                if (i - j >= 0) dp[i] += dp[i - j];
            }
        }
        return dp[n];
    }
};

爬楼梯的最小花费

思路：其实主要就是对于动归数组的定义和相关递推公式的研究，以及关键的爬楼梯时的第一、二步的花费相关的问题。

这里：dp[i]=min(dp[i-1]+cost[i-1],dp[i-2]+cost[i-2]);

同时，在题目要求中，所写的初始在第0层的时候可以不花费，或者直接从第一楼开始也行，所以这两层楼都不花费，同时，由这两层楼可以推导出后续所有楼的最小花费。同时，递推顺序，很明显是从前往后，所以这里就直接for循环即可，但是在遇到比如背包问题的时候，我们需要重新考虑，因为背包问题需要两个for循环，因此在考虑循环的时候要重点关注for循环的遍历顺序。但是在本题中无需过多关注该问题。

代码展示：

int minCostClimbingStairs(vector<int>& cost) {
        //if(cost.size()<=1) return cost[0];
        int dp0=0;
        int dp1=0;
        for(int i=2;i<=cost.size();i++){
            int sum=min(dp0+cost[i-2],dp1+cost[i-1]);
            dp0=dp1;
            dp1=sum;
        }
        return dp1;
    }
};

总结，总体上该问题是对前面爬楼梯问题的一个延续和变形，因此我们在计算的时候，需要进一步考虑每层楼的最小花费，因此需要用到min函数，同时我们可以根据相关条件，进行相关操作。
