    Bellman_ford算法¶

    题目来源：https://kamacoder.com/problempage.php?pid=1152

    简单描述：

    某国为促进城市间经济交流，决定对货物运输提供补贴。共有 n 个编号为 1 到 n 的城市，通过道路网络连接，网络中的道路仅允许从某个城市单向通行到另一个城市，不能反向通行。

    网络中的道路都有各自的运输成本和政府补贴，道路的权值计算方式为：运输成本 - 政府补贴。

    权值为正表示扣除了政府补贴后运输货物仍需支付的费用；权值为负则表示政府的补贴超过了支出的运输成本，实际表现为运输过程中还能赚取一定的收益。

    请找出从城市 1 到城市 n 的所有可能路径中，综合政府补贴后的最低运输成本。

    如果最低运输成本是一个负数，它表示在遵循最优路径的情况下，运输过程中反而能够实现盈利。

    城市 1 到城市 n 之间可能会出现没有路径的情况，同时保证道路网络中不存在任何负权回路。

    负权回路是指一系列道路的总权值为负，这样的回路使得通过反复经过回路中的道路，理论上可以无限地减少总成本或无限地增加总收益。

    输入描述

    第一行包含两个正整数，第一个正整数 n 表示该国一共有 n 个城市，第二个整数 m 表示这些城市中共有 m 条道路。

    接下来为 m 行，每行包括三个整数，s、t 和 v，表示 s 号城市运输货物到达 t 号城市，道路权值为 v（单向图）。

    输出描述

    如果能够从城市 1 到连通到城市 n， 请输出一个整数，表示运输成本。如果该整数是负数，则表示实现了盈利。如果从城市 1 没有路径可达城市 n，请输出 "unconnected"。

    思路¶

    这种类型的题目主要是求解一个起点到某个终点的最短路径。

    1、在本题中，求解的是从1->n的最短路径，注意在这些题目中有些边是负权值，因此不能使用dijkstra算法。

    因为dijkstra算法，他主要是每次选择权值最小的边，但是如果路径过程中存在一个负权值的边，那么在求解最短路径的过程中如果出现负数，那么我们在选择的时候可能会错过出现负数的边，最后导致计算的最短路径出现错误。

    2、bellman_ford算法的主要思路其实是利用了动态规划的思想，使用minDist[B] = min(minDist[A] + value, minDist[B])

    通过将整体问题分解为多步子序列进行求解，从而得到答案。

    上面的那个公式在算法中被定义为松弛。

    每一次松弛都相当于求解了一次从1到一条边的长度的所有最短路径，第二次松弛，就会得到长度为2的最短路径，那么我们要求解1-n的最短路径，我们需要n-1次松弛。

    代码：

    C++
    #include <iostream>
#include <vector>
#include <list>
#include <climits>
using namespace std;

int main() {
    int n, m, p1, p2, val;
    cin >> n >> m;

    vector<vector<int>> grid;

    // 将所有边保存起来
    for(int i = 0; i < m; i++){
        cin >> p1 >> p2 >> val;
        // p1 指向 p2，权值为 val
        grid.push_back({p1, p2, val});

    }
    int start = 1;  // 起点
    int end = n;    // 终点

    vector<int> minDist(n + 1 , INT_MAX);
    minDist[start] = 0;
    for (int i = 1; i < n; i++) { // 对所有边 松弛 n-1 次
        for (vector<int> &side : grid) { // 每一次松弛，都是对所有边进行松弛
            int from = side[0]; // 边的出发点
            int to = side[1]; // 边的到达点
            int price = side[2]; // 边的权值
            // 松弛操作 
            // minDist[from] != INT_MAX 防止从未计算过的节点出发
            if (minDist[from] != INT_MAX && minDist[to] > minDist[from] + price) { 
                minDist[to] = minDist[from] + price;  
            }
        }
    }
    if (minDist[end] == INT_MAX) cout << "unconnected" << endl; // 不能到达终点
    else cout << minDist[end] << endl; // 到达终点最短路径

}

    SPFA算法¶

    题目来源：https://kamacoder.com/problempage.php?pid=1152

    题目描述和上一个题目一样。

    思路¶

    该算法主要是为了减少循环次数，我们只有在更新过的数组元素，下一次我们才会进行松弛。

    因此我们使用一个队列来存储更新过的结点。

    每一次如果我们更新了mindist[x]那么我们更新元素的同时，将该元素加入到队列中，只要队列中还有元素我们就一直循环处理。

    我们如何存放开始时输入的边呢？

    可以使用结构体数组来存放元素，然后构造一个邻接表，

    Text Only
     struct Edge { //邻接表
    int to;  // 链接的节点
    int val; // 边的权重

    Edge(int t, int w): to(t), val(w) {}  // 构造函数
};
 vector<list<Edge>> grid(n + 1);

    上述的部分定义完成后，我们就可以直接根据思路来编写代码

    代码：

    C++
    #include <iostream>
#include <vector>
#include <queue>
#include <list>
#include <climits>
using namespace std;

struct Edge { //邻接表
    int to;  // 链接的节点
    int val; // 边的权重

    Edge(int t, int w): to(t), val(w) {}  // 构造函数
};


int main() {
    int n, m, p1, p2, val;
    cin >> n >> m;

    vector<list<Edge>> grid(n + 1); 

    vector<bool> isInQueue(n + 1); // 加入优化，已经在队里里的元素不用重复添加

    // 将所有边保存起来
    for(int i = 0; i < m; i++){
        cin >> p1 >> p2 >> val;
        // p1 指向 p2，权值为 val
        grid[p1].push_back(Edge(p2, val));
    }
    int start = 1;  // 起点
    int end = n;    // 终点

    vector<int> minDist(n + 1 , INT_MAX);
    minDist[start] = 0;

    queue<int> que;
    que.push(start); 

    while (!que.empty()) {

        int node = que.front(); que.pop();
        isInQueue[node] = false; // 从队列里取出的时候，要取消标记，我们只保证已经在队列里的元素不用重复加入
        for (Edge edge : grid[node]) {
            int from = node;
            int to = edge.to;
            int value = edge.val;
            if (minDist[to] > minDist[from] + value) { // 开始松弛
                minDist[to] = minDist[from] + value; 
                if (isInQueue[to] == false) { // 已经在队列里的元素不用重复添加
                    que.push(to);
                    isInQueue[to] = true;
                }
            }
        }

    }
    if (minDist[end] == INT_MAX) cout << "unconnected" << endl; // 不能到达终点
    else cout << minDist[end] << endl; // 到达终点最短路径
}

    Bellman_ford之判断负权回路¶

    题目来源：https://kamacoder.com/problempage.php?pid=1153

    题目描述和上面的题目一样。

    思路¶

    这个队列优化算法主要是解决整个图中存在一条负权回路的情况，在上面的初始算法的时候，如果图中存在一个负权回路，那么我们在每次进行松弛操作的时候都会给mindist叠加负值，最后理论上能够叠加到负无穷大，所以我们需要处理这种情况。

    在本题中我们如果找到一个负权回路，那么我们就可以输出circle，如果没有路径就输出unconnected，剩下的情况就可以直接输出对应的负权回路了。

    如何判断负权回路？

    因为出现负权回路的时候，每多一次循环，那么对应的那个负权回路的mindist就会发生变化，需要对其进行更新。

    但是如果没有负权回路则不会发生任何变化。

    因为我们正常是使用n-1松弛来操作相关的循环即可，所以我们在n-1的基础上增加一次循环即可，通过这次循环，如果需要更新，那么表示有负权回路。但是我们判断的时候，不能真的对其进行更新，所以需要在循环过程中单独处理第n次松弛。

    具体实现相关代码如下：

    C++
    #include <iostream>
#include <vector>
#include <list>
#include <climits>
using namespace std;

int main() {
    int n, m, p1, p2, val;
    cin >> n >> m;

    vector<vector<int>> grid;

    for(int i = 0; i < m; i++){
        cin >> p1 >> p2 >> val;
        // p1 指向 p2，权值为 val
        grid.push_back({p1, p2, val});

    }
    int start = 1;  // 起点
    int end = n;    // 终点

    vector<int> minDist(n + 1 , INT_MAX);
    minDist[start] = 0;
    bool flag = false;
    for (int i = 1; i <= n; i++) { // 这里我们松弛n次，最后一次判断负权回路
        for (vector<int> &side : grid) {
            int from = side[0];
            int to = side[1];
            int price = side[2];
            if (i < n) {
                if (minDist[from] != INT_MAX && minDist[to] > minDist[from] + price) minDist[to] = minDist[from] + price;
            } else { // 多加一次松弛判断负权回路
                if (minDist[from] != INT_MAX && minDist[to] > minDist[from] + price) flag = true;

            }
        }

    }

    if (flag) cout << "circle" << endl;
    else if (minDist[end] == INT_MAX) {
        cout << "unconnected" << endl;
    } else {
        cout << minDist[end] << endl;
    }
}

    bellman_ford之单源有限最短路¶

    题目来源：https://kamacoder.com/problempage.php?pid=1154

    简单描述：

    和上面几个题的思路大差不差，主要的区别是在最多经过 k 个城市的条件下，计算从城市 src 到城市 dst 的最低运输成本。

    思路¶

    这个题目的限制很简单，在前面的题目中我们的松弛次数依赖于起点和终点的最长距离，即n-1次，但是在该题目中我们限制了距离长度，所以我们只需要将之前的松弛n-1次修改为k-1次即可。

    要注意计算minDist数组的时候，基于了本次松弛的 minDist数值，而不是上一次 松弛时候minDist的数值。 所以在每次计算 minDist 时候，要基于 对所有边上一次松弛的 minDist 数值才行，所以我们要记录上一次松弛的minDist。

    代码：

    C++
    #include <iostream>
#include <vector>
#include <list>
#include <climits>
using namespace std;

int main() {
    int src, dst,k ,p1, p2, val ,m , n;

    cin >> n >> m;

    vector<vector<int>> grid;

    for(int i = 0; i < m; i++){
        cin >> p1 >> p2 >> val;
        grid.push_back({p1, p2, val});
    }

    cin >> src >> dst >> k;

    vector<int> minDist(n + 1 , INT_MAX);
    minDist[src] = 0;
    vector<int> minDist_copy(n + 1); // 用来记录上一次遍历的结果
    for (int i = 1; i <= k + 1; i++) {
        minDist_copy = minDist; // 获取上一次计算的结果
        for (vector<int> &side : grid) {
            int from = side[0];
            int to = side[1];
            int price = side[2];
            // 注意使用 minDist_copy 来计算 minDist 
            if (minDist_copy[from] != INT_MAX && minDist[to] > minDist_copy[from] + price) {  
                minDist[to] = minDist_copy[from] + price;
            }
        }
    }
    if (minDist[dst] == INT_MAX) cout << "unreachable" << endl; // 不能到达终点
    else cout << minDist[dst] << endl; // 到达终点最短路径

}
