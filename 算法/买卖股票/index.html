    买卖股票¶

    主要是对代码随想录刷题的相关思路总和和回顾，仅用于个人学习用途。

    最佳时机1¶

    题目来源：https://leetcode.cn/problems/best-time-to-buy-and-sell-stock/

    思路：给定了一个数组prices，第i个元素表示一只给定的股票第I天的价格。

    要求：最大利润。

    只能买卖一次。

    使用动态规划：

    1、确定dp数组的含义：dp[i][0]表示第I天的时候，就持有股票时的当前的所得最多现金。dp[i][1]表示第i天的时候，不持有股票的所得最多现金。

    2、dp数组的推导公式，每一天都有两种状态，持有股票和不持有股票，

    持有股票的时候分为两种情况，一种是今天之前就买过股票了，今天不用管，直接使用前一天的现金金额即可，另一种是前一天没买，今天才刚买，只需要计算dp[i-1][1]-prices[i]即可

    所以，dp[i][0]=max(dp[i-1][0],-prices[i]);

    不持有股票也分为两种情况，一种是今天之前已经卖过了，一种是今天刚卖。

    dp[i][1]=max(dp[i-1][1],dp[i-1][0]+prices[i]);

    3、dp数组的初始化，很明显，每个数组都依赖于前一个数组元素，因此我们只需要初始化dp[0][0]和dp[0][1]即可。dp[0][0]表示第零天持有股票，dp[0][0]-=prices[0],dp[0][1]=0，第0天不持有股票代表着当天的股票已经当天买当天卖了，因此为0.

    4、dp数组的推导顺序，我们根据推导公式可以看出来是根据前后推导的，因此我们可以只用一个for循环即可。

    5、举例推导，自行推导即可。

    代码：

    C++
    int maxProfit(vector<int>& prices) {
        int len = prices.size();
        if (len == 0) return 0;
        vector<vector<int>> dp(len, vector<int>(2));
        dp[0][0] -= prices[0];
        dp[0][1] = 0;
        for (int i = 1; i < len; i++) {
            dp[i][0] = max(dp[i - 1][0], -prices[i]);
            dp[i][1] = max(dp[i - 1][1], prices[i] + dp[i - 1][0]);
        }
        return dp[len - 1][1];
    }

    买卖股票的最佳时机2¶

    题目来源：https://leetcode.cn/problems/best-time-to-buy-and-sell-stock-ii/

    题目简单描述：

    计算最大利润，同时尽可能完成更多交易，每次购买股票之前都得保证之前的股票已经卖了。

    思路：

    其实和上一个题目的思路类似，但是，唯一区别的地方就是每个股票持有的时候，可能会有前一天的所剩余的现金，因此只有这个地方有不一样的东西。

    也就是dp[i][0]=max(dp[i-1][0],dp[i-1][1]-prices[i]);

    代码：

    C++
    int maxProfit(vector<int>& prices) {
        int len = prices.size();
        vector<vector<int>> dp(len, vector<int>(2, 0));
        dp[0][0] -= prices[0];
        dp[0][1] = 0;
        for (int i = 1; i < len; i++) {
            dp[i][0] = max(dp[i - 1][0], dp[i - 1][1] - prices[i]); // 注意这里是和121. 买卖股票的最佳时机唯一不同的地方。
            dp[i][1] = max(dp[i - 1][1], dp[i - 1][0] + prices[i]);
        }
        return dp[len - 1][1];
    }

    减少空间使用率的解法，其实就是每次计算的时候只涉及两个数组元素，且最后求解的时候是选择最后一个数组元素的，所以可以使用大小为2*2的动态数组。

    代码：

    C++
    int maxProfit(vector<int>& prices) {
        int len = prices.size();
        vector<vector<int>> dp(2, vector<int>(2));
        dp[0][0] -= prices[0];
        dp[0][1] = 0;
        for (int i = 1; i < len; i++) {
            dp[i % 2][0] = max(dp[(i - 1) % 2][0], dp[(i - 1) % 2][1] - prices[i]);
            dp[i % 2][1] = max(dp[(i - 1) % 2][1], prices[i] + dp[(i - 1) % 2][0]);
        }
        return dp[(len - 1) % 2][1];
    }

    买卖股票3¶

    题目来源：https://leetcode.cn/problems/best-time-to-buy-and-sell-stock-iii/

    简单描述：

    该题限制你的交易次数，限制为2两次，而且不能同时参与多个交易，因此每次交易之前必须先卖掉之前持有的股票。

    思路：

    关键其实在于至多买卖两次，因此可以买卖一次也可以买卖两次。

    此时需要考虑的状态就有很多。

    0、什么也不干

    1、第一次持有股票

    2、第一次不持有股票

    3、第二次持有股票

    4、第二次不持有股票

    dp[i][1]=max(dp[i-1][0] - prices[i], dp[i - 1][1]);

    dp[i][2]=max(dp[i-1][2],dp[i][1]+prices[i]);

    dp[i][3]=max(dp[i-1][2]-prices[i],dp[i-1][3]);

    dp[i][4]=max(dp[i-1][3]+prices[i],dp[i-1][4]);

    数组初始化：dp[0][0]=0,dp[0][1]=-prices[0],dp[0][3]=-prices[0];

    确定遍历顺序，很明显从前往后，因此可以直接遍历即可。

    代码：

    C++
    int maxProfit(vector<int>& prices) {
        if (prices.size() == 0) return 0;
        vector<vector<int>> dp(prices.size(), vector<int>(5, 0));
        dp[0][1] = -prices[0];
        dp[0][3] = -prices[0];
        for (int i = 1; i < prices.size(); i++) {
            dp[i][0] = dp[i - 1][0];
            dp[i][1] = max(dp[i - 1][1], dp[i - 1][0] - prices[i]);
            dp[i][2] = max(dp[i - 1][2], dp[i - 1][1] + prices[i]);
            dp[i][3] = max(dp[i - 1][3], dp[i - 1][2] - prices[i]);
            dp[i][4] = max(dp[i - 1][4], dp[i - 1][3] + prices[i]);
        }
        return dp[prices.size() - 1][4];
    }

    空间优化版本：

    C++
     int maxProfit(vector<int>& prices) {
        if (prices.size() == 0) return 0;
        vector<int> dp(5, 0);
        dp[1] = -prices[0];
        dp[3] = -prices[0];
        for (int i = 1; i < prices.size(); i++) {
            dp[1] = max(dp[1], dp[0] - prices[i]);
            dp[2] = max(dp[2], dp[1] + prices[i]);
            dp[3] = max(dp[3], dp[2] - prices[i]);
            dp[4] = max(dp[4], dp[3] + prices[i]);
        }
        return dp[4];
    }

    买卖股票4¶

    题目来源:https://leetcode.cn/problems/best-time-to-buy-and-sell-stock-iv/

    简单描述：

    设置最多只能完成k笔交易

    不能同时参加多笔交易。

    思路：

    和上一题的思路类似

    dp[i][j]表示第i天，j表示某次卖出买入操作。

    j的状态表示为：

    • 0 表示不操作
    • 1 第一次买入
    • 2 第一次卖出
    • 3 第二次买入
    • 4 第二次卖出

    ...

    然后直接在原来一层循环中，加上一层对于j的循环即可。

    代码：

    C++
    int maxProfit(int k, vector<int>& prices) {

        if (prices.size() == 0) return 0;
        vector<vector<int>> dp(prices.size(), vector<int>(2 * k + 1, 0));
        for (int j = 1; j < 2 * k; j += 2) {
            dp[0][j] = -prices[0];
        }
        for (int i = 1;i < prices.size(); i++) {
            for (int j = 0; j < 2 * k - 1; j += 2) {
                dp[i][j + 1] = max(dp[i - 1][j + 1], dp[i - 1][j] - prices[i]);
                dp[i][j + 2] = max(dp[i - 1][j + 2], dp[i - 1][j + 1] + prices[i]);
            }
        }
        return dp[prices.size() - 1][2 * k];
    }

    最佳买卖股票时机含冷冻期¶

    题目来源：https://leetcode.cn/problems/best-time-to-buy-and-sell-stock-with-cooldown/

    简单描述：

    设计一个算法计算出最大利润。在满足以下约束条件下，你可以尽可能地完成更多的交易（多次买卖一支股票）:

    • 你不能同时参与多笔交易（你必须在再次购买前出售掉之前的股票）。
    • 卖出股票后，你无法在第二天买入股票 (即冷冻期为 1 天)。

    思路：

    该题需要设置的dp数组需要考虑四种状态。

    1、持有股票状态

    2、不持有股票状态——今天卖出股票

    3、不持有股票状态——保持卖出股票的状态（一种是两天之前就卖出了，超过了一天冷冻期，一种是前一天才卖，一直没操作）

    4、当天为冷冻期，持续一天，当天没有操作

    递推公式：

    状态1：

    dp[i][0]=max(dp[i-1][0],dp[i-1][3]-prices[i],dp[i-1][2]-prices[i]);

    状态2：

    dp[i][1]=dp[i-1][0]+prices[i]

    状态3：

    dp[i][2]=max(dp[i-1][2],dp[i-1][3]);

    状态4：

    dp[i][3]=dp[i-1][1]

    递推顺序：很明显从前往后。

    初始化：

    dp[0][0]=-prices[0];

    代码：

    C++
    int maxProfit(vector<int>& prices) {
        int n = prices.size();
        if (n == 0) return 0;
        vector<vector<int>> dp(n, vector<int>(4, 0));
        dp[0][0] -= prices[0]; // 持股票
        for (int i = 1; i < n; i++) {
            dp[i][0]=max(dp[i-1][0],dp[i-1][3]-prices[i],dp[i-1][2]-prices[i]);
            dp[i][1]=dp[i-1][0]+prices[i];
            dp[i][2]=max(dp[i-1][2],dp[i-1][3]);
            dp[i][3]=dp[i-1][1];
        }
        return max(dp[n - 1][3], max(dp[n - 1][1], dp[n - 1][2]));
    }

    买卖股票的最佳时机含手续费¶

    题目来源：https://leetcode.cn/problems/best-time-to-buy-and-sell-stock-with-transaction-fee/

    简单描述：

    你可以无限次地完成交易，但是你每笔交易都需要付手续费。如果你已经购买了一个股票，在卖出它之前你就不能再继续购买股票了。

    返回获得利润的最大值。

    注意：这里的一笔交易指买入持有并卖出股票的整个过程，每笔交易你只需要为支付一次手续费。

    思路：

    其实这个题目的思路和买卖股票2的思路差不多，主要差别在于需要计算手续费。

    因此这里直接给出代码，进行相关的理解即可。

    代码：

    C++
    int maxProfit(vector<int>& prices, int fee) {
        int n = prices.size();
        vector<vector<int>> dp(n, vector<int>(2, 0));
        dp[0][0] -= prices[0]; // 持股票
        for (int i = 1; i < n; i++) {
            dp[i][0] = max(dp[i - 1][0], dp[i - 1][1] - prices[i]);
            dp[i][1] = max(dp[i - 1][1], dp[i - 1][0] + prices[i] - fee);
        }
        return max(dp[n - 1][0], dp[n - 1][1]);
    }
