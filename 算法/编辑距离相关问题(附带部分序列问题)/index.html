    最长公共子序列¶

    题目来源：https://leetcode.cn/problems/longest-common-subsequence/

    简单描述：

    给定两个字符串 text1 和 text2，返回这两个字符串的最长公共子序列的长度。

    一个字符串的 子序列 是指这样一个新的字符串：它是由原字符串在不改变字符的相对顺序的情况下删除某些字符（也可以不删除任何字符）后组成的新字符串。

    例如，"ace" 是 "abcde" 的子序列，但 "aec" 不是 "abcde" 的子序列。两个字符串的「公共子序列」是这两个字符串所共同拥有的子序列。

    若这两个字符串没有公共子序列，则返回 0。

    思路：

    该题目使用动态规划的解法。

    1、确定dp数组的含义，dp[i][j]表示以i-1，j-1为结尾的字符串的最长公共子序列。

    2、dp数组的递推公式：

    一种是text1[i-1]=text2[j-1]，此时dp[i][j]=dp[i-1][j-1]+1;

    一种是text1[i-1]!=text2[j-1]，那么一种是0-i-2的字符串和0-j-1的最长公共子序列长度。

    一种是0-i-1,0-j-2的最长公共子序列长度，二者取最大值即可。

    3、dp数组初始化，dp[i][0]=0,dp[0][j]=0;其他的数组元素的值都依赖于这两列元素。

    4、确定遍历顺序，因为每个元素都依赖于左上角的元素或者左边，上边的元素，因此遍历顺序我们可以很轻易的得到，是从上往下，从左往右。

    5、举例推导，自行举例进行相关推导即可。

    代码：

    C++
    int longestCommonSubsequence(string text1, string text2) {
        vector<vector<int>> dp(text1.size() + 1, vector<int>(text2.size() + 1, 0));
        for (int i = 1; i <= text1.size(); i++) {
            for (int j = 1; j <= text2.size(); j++) {
                if (text1[i - 1] == text2[j - 1]) {
                    dp[i][j] = dp[i - 1][j - 1] + 1;
                } else {
                    dp[i][j] = max(dp[i - 1][j], dp[i][j - 1]);
                }
            }
        }
        return dp[text1.size()][text2.size()];
    }

    不相交的线¶

    题目来源：https://leetcode.cn/problems/uncrossed-lines/

    简单描述：

    我们在两条独立的水平线上按给定的顺序写下 A 和 B 中的整数。

    现在，我们可以绘制一些连接两个数字 A[i] 和 B[j] 的直线，只要 A[i] == B[j]，且我们绘制的直线不与任何其他连线（非水平线）相交。

    以这种方法绘制线条，并返回我们可以绘制的最大连线数。

    思路：

    我们仔细分析，其实我们可以很明显看出来，其实求解的就是两个序列中的按顺序的最长公共子序列长度，而且二者不能相交，那就说明必须是子序列，因为子序列是按顺序的。

    最大子序和¶

    题目来源：https://leetcode.cn/problems/maximum-subarray/

    简单描述：

    给定一个整数数组 nums ，找到一个具有最大和的连续子数组（子数组最少包含一个元素），返回其最大和。

    思路：

    1、确定dp数组及其含义，dp[i]：包括下标i的最大连续子序列和为dp[i]

    2、确定递推公式，当nums[i]加入到子序列和的时候，判断原来的dp[i]和新的dp[i]哪个大，如果大，就加入，不大就不变。

    3、dp数组的初始化，因为我们求解的最大和，所以我们只需要将所有元素的和初始化为0即可，dp[0]=nums[0]即可。

    4、确定遍历顺序，从前往后

    5、举例推导，自行推导。

    代码：

    C++
    int maxSubArray(vector<int>& nums) {
        if (nums.size() == 0) return 0;
        vector<int> dp(nums.size());
        dp[0] = nums[0];
        int result = dp[0];
        for (int i = 1; i < nums.size(); i++) {
            dp[i] = max(dp[i - 1] + nums[i], nums[i]); // 状态转移公式
            if (dp[i] > result) result = dp[i]; // result 保存dp[i]的最大值
        }
        return result;
    }

    判断子序列¶

    题目来源：https://leetcode.cn/problems/is-subsequence/

    简单描述：

    给定字符串 s 和 t ，判断 s 是否为 t 的子序列。

    字符串的一个子序列是原始字符串删除一些（也可以不删除）字符而不改变剩余字符相对位置形成的新字符串。（例如，"ace"是"abcde"的一个子序列，而"aec"不是）。

    思路：

    1、确定dp数组的含义，dp[i][j]表示以i-1为结尾的字符串s，和以j-1为结尾的字符串t的最长公共子序列。

    2、确定递推公式，

    • if (s[i - 1] == t[j - 1])
    • t中找到了一个字符在s中也出现了
    • if (s[i - 1] != t[j - 1])
    • 相当于t要删除元素，继续匹配

    3、dp数组的初始化

    从递推公式中得出，每个元素都依赖于前面的元素，同时是求解最长公共子序列长度，所以直接dp[i][0]和dp[0][j]其实就表示0.

    4、确定遍历顺序

    从前往后，即从上往下，从左往右。

    5、举例推导dp数组，自行推导即可。

    代码：

    C++
    bool isSubsequence(string s, string t) {
        vector<vector<int>> dp(s.size() + 1, vector<int>(t.size() + 1, 0));
        for (int i = 1; i <= s.size(); i++) {
            for (int j = 1; j <= t.size(); j++) {
                if (s[i - 1] == t[j - 1]) dp[i][j] = dp[i - 1][j - 1] + 1;
                else dp[i][j] = dp[i][j - 1];
            }
        }
        if (dp[s.size()][t.size()] == s.size()) return true;
        return false;
    }

    不同的子序列¶

    题目来源：https://leetcode.cn/problems/distinct-subsequences/

    简单描述：

    给定一个字符串 s 和一个字符串 t ，计算在 s 的子序列中 t 出现的个数。

    字符串的一个 子序列 是指，通过删除一些（也可以不删除）字符且不干扰剩余字符相对位置所组成的新字符串。（例如，"ACE" 是 "ABCDE" 的一个子序列，而 "AEC" 不是）

    题目数据保证答案符合 32 位带符号整数范围。

    思路：

    1、确定dp数组的含义，dp[i][j]表示以i-1,j-1为标的两个字符串的子序列出现的个数。

    2、确定递推公式，分为两种情况，s[i-1]=t[j-1]和两者不等两种情况。

    如果相等，那么dp[i][j] = dp[i - 1][j - 1] + dp[i - 1][j];

    如果不相等，那么dp[i][j] = dp[i - 1][j];

    3、dp数组怎么初始化，

    很明显，我们可以得出，数组初始化在dp[i][0]的时候，表示字符串s中包含空字符串的子序列的个数为1，因此都初始化为1，

    但是在dp[0][j]中不存在空字符串包含某些字符串，因此为0，同时dp[0][0]表示为1.

    4、递推顺序，从上到下，从左到右。

    5、举例推导

    代码：

    C++
    int numDistinct(string s, string t) {
        vector<vector<uint64_t>> dp(s.size() + 1, vector<uint64_t>(t.size() + 1));
        for (int i = 0; i < s.size(); i++) dp[i][0] = 1;
        for (int j = 1; j < t.size(); j++) dp[0][j] = 0;
        for (int i = 1; i <= s.size(); i++) {
            for (int j = 1; j <= t.size(); j++) {
                if (s[i - 1] == t[j - 1]) {
                    dp[i][j] = dp[i - 1][j - 1] + dp[i - 1][j];
                } else {
                    dp[i][j] = dp[i - 1][j];
                }
            }
        }
        return dp[s.size()][t.size()];
    }

    两个字符串的删除操作¶

    题目来源：https://leetcode.cn/problems/delete-operation-for-two-strings/

    简单描述：

    给定两个单词 word1 和 word2，找到使得 word1 和 word2 相同所需的最小步数，每步可以删除任意一个字符串中的一个字符。

    代码：

    C++
    int minDistance(string word1, string word2) {
        vector<vector<int>> dp(word1.size() + 1, vector<int>(word2.size() + 1));
        for (int i = 0; i <= word1.size(); i++) dp[i][0] = i;
        for (int j = 0; j <= word2.size(); j++) dp[0][j] = j;
        for (int i = 1; i <= word1.size(); i++) {
            for (int j = 1; j <= word2.size(); j++) {
                if (word1[i - 1] == word2[j - 1]) {
                    dp[i][j] = dp[i - 1][j - 1];
                } else {
                    dp[i][j] = min(dp[i - 1][j] + 1, dp[i][j - 1] + 1);
                }
            }
        }
        return dp[word1.size()][word2.size()];
    }

    代码2：

    C++
    int minDistance(string word1, string word2) {
        vector<vector<int>> dp(word1.size()+1, vector<int>(word2.size()+1, 0));
        for (int i=1; i<=word1.size(); i++){
            for (int j=1; j<=word2.size(); j++){
                if (word1[i-1] == word2[j-1]) dp[i][j] = dp[i-1][j-1] + 1;
                else dp[i][j] = max(dp[i-1][j], dp[i][j-1]);
            }
        }
        return word1.size()+word2.size()-dp[word1.size()][word2.size()]*2;
    }

    编辑距离¶

    题目来源：https://leetcode.cn/problems/edit-distance/

    简单描述：

    给你两个单词 word1 和 word2，请你计算出将 word1 转换成 word2 所使用的最少操作数 。

    你可以对一个单词进行如下三种操作：

    • 插入一个字符
    • 删除一个字符
    • 替换一个字符

    思路：

    1、确定dp数组及其相关含义

    dp[i][j] 表示以下标i-1为结尾的字符串word1，和以下标j-1为结尾的字符串word2，最最少操作数为dp[i][j]。

    2、确定递推公式

    如果word1[i-1]==word2[j-1]，那么我们就不操作

    如果不相等，那我们就考虑当前位置的元素是删除还是修改还是增加，其实删除和增加是差不多的，增加相当于前面的元素删掉了，+1,。

    3、确定dp数组的初始化，

    dp[i][0] ：以下标i-1为结尾的字符串word1，和空字符串word2，最近编辑距离为dp[i][0]。

    那么dp[i][0]就应该是i，对word1里的元素全部做删除操作，即：dp[i][0] = i;

    同理dp[0][j] = j;

    4、遍历顺序

    从左往右，从上到下

    5、举例推导

    代码：

    C++
    int minDistance(string word1, string word2) {
        vector<vector<int>> dp(word1.size() + 1, vector<int>(word2.size() + 1, 0));
        for (int i = 0; i <= word1.size(); i++) dp[i][0] = i;
        for (int j = 0; j <= word2.size(); j++) dp[0][j] = j;
        for (int i = 1; i <= word1.size(); i++) {
            for (int j = 1; j <= word2.size(); j++) {
                if (word1[i - 1] == word2[j - 1]) {
                    dp[i][j] = dp[i - 1][j - 1];
                }
                else {
                    dp[i][j] = min({dp[i - 1][j - 1], dp[i - 1][j], dp[i][j - 1]}) + 1;
                }
            }
        }
        return dp[word1.size()][word2.size()];
    }
